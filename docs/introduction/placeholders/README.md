# Screenshot Placeholders
**Tags:** repo:autovim area:introduction type:reference owner:shared
**Abstract:** Sample buffers to open in Neovim when capturing live AutoVim screenshots for the introduction page.

These files are intentionally small and polished so the editor side of a screenshot has useful content while plugin panels are open.

Suggested live capture setup:

1. Open [sample-service.go](sample-service.go) in the main editor.
2. Open [agent-review-note.md](agent-review-note.md) in a second split or Markdown render slot.
3. Open [sample-api.http](sample-api.http) when capturing the Kulala workflow.
4. Open [worktree-plan.md](worktree-plan.md) when capturing worktree, Git graph, or auto-agent KB flows.

The generated assets in `../assets/` are illustrative. These placeholders are for a future true screenshot/GIF capture pass.
